# Data Cleaning and Data Quality

This repository contains notebooks and datasets related to **data cleaning and data quality analysis**.

## Contents

**DataCleaning.ipynb**
Demonstrates common techniques used to clean raw datasets.

**Data_Quality_Check_Accuracy_Completeness.ipynb**
Explains how to evaluate data quality using metrics such as accuracy and completeness.

**Datasets**

* DataScience_Youtube_Videos.csv
* Data_Quality.csv

These datasets are used in the notebooks for demonstration.
